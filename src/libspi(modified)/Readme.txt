INSTALL

Put libspi.a in the lib folder of libnds. (for instance C:\devkitPro\libnds\lib)
Put spi.h in the include folder of libnds. (for instance C:\devkitPro\libnds\include)

See the example source code on how to use the library

Check www.circuitdb.com for the latest updates (seems no longer valid)
